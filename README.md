# mongodb-crud

| Route           | HTTP          | Description  |
| --------------- |:-------------:| -------------:|
| /      | GET           | get all books info  |
| /:id  | GET           |   get a single book info |
| /      | POST          |    create a book info   |
| /:id  | DELETE        |    delete a book info  |
| /:id  | PUT           |    update a book  with new info  |

### USAGE
```
$npm init
$npm mongodb express
$ npm run dev
